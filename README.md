# 人味儿写作 · Renwei Writing

**中文** | [English](README.en.md)

> 改完之后，那个人还在。

一个 AI agent skill，用于打磨、润色、改写别人的文字时，保住文字背后那个人的存在感。

诞生于一次真实的失败：一段好文案被 AI 磨了三遍，一遍比一遍"漂亮"，一遍比一遍没有人。作者说：

> "AI 改东西，缺乏一个存在感，我感受不到背后的那个人。"

## 核心观点

人味儿不是文字技巧。它是三件事：

- **位置** — 一个具体的人站在具体的位置说话。AI 不站在任何地方，这种无限灵活反过来就是没有存在。
- **代价** — 真实的句子背后有真实的观察和经历。读者闻得出来哪句话背后有生活，哪句话背后只有词库。
- **手迹** — 看似冗余的语气词、不工整的呼吸，都是这个人说话的样子。AI 没有手，怎么写都像打印的。

## 操作规则

1. 只做减法，少动。动三处是常态，动十处是事故
2. 毛边先假设是手迹，不是瑕疵
3. 不写金句。改稿时写出漂亮句子是警报：那是你在表演，不是他在说话
4. 逐处交代改动，把否决权留给作者
5. 上限是隐形

```bash
git clone https://github.com/orange2ai/renwei-writing.git ~/.cola/skills/renwei-writing
```

或将整个目录放入你的 agent 的 skills 路径。

## 与风格规范类 skill 的关系

禁用词清单（破折号、"不是X而是Y"、AI tropes）管的是不要写出 AI 的句式。本 skill 管的是不要抹掉人的存在。先用本 skill 决定"动不动"，再用风格规范检查动过的地方。
